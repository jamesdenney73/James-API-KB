# API Expert Knowledge Base

## Overview

This Knowledge Base is designed for an **API Expert Agent** that helps vibe coders build API-driven integrations across three platforms:

- **Hatz AI** - AI/ML platform with chat completions, agents, workflows, and app builder
- **Go High Level** - Marketing automation CRM with contacts, conversations, and webhooks
- **n8n** - Workflow automation platform with visual node-based programming
